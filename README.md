### 🌟 Featured Projects

#### 🍳 [Freezer-to-Feast](https://github.com/anuraj07/FreezerToFeast) — Fresh Harvest Meal Tracker
> **A premium, serverless Android application designed with a "Quiet Luxury" aesthetic.** It helps users track kitchen parameters, customize dietary choices, log daily meals, and leverage client-side Generative AI (`gemini-2.5-flash`) via the Firebase Vertex AI SDK to suggest tailored recipes based on available ingredients.
- **Tech Stack:** Kotlin, Jetpack Compose, MVVM, SQLite / Room, Firebase (Auth, Firestore), Vertex AI Firebase SDK.
- **Highlights:** Dynamic AI Chef JSON generator, real-time Firestore sync with strict YYYY-MM-DD log security validations.

#### 🗺️👀 [Sideglance](https://github.com/anuraj07/Sideglance) — Real-Time Presence & Status Sharing
> **A minimal, modern Jetpack Compose partner connection app.** Partners can securely pair via CameraX/ZXing QR scan or deep links and share their real-time location, activity status (Work, Cook, Play, Rest, etc.), and relative distance calculated on-device.
- **Tech Stack:** Kotlin, Jetpack Compose (Material 3), Firebase Auth & Firestore, Jetpack Glance Widgets, WorkManager API, CameraX & ZXing.
- **Highlights:** Haversine formula distance calculations with location opt-out privacy fallback (`∞`), background Glance home widgets, and foreground status change notifications.

#### 📝 [Notify](https://github.com/anuraj07/notify) — Modern Note-Taking App
> **A clean, offline-first note-taking application designed with an "Organic Professional" visual identity.** Features clean Forest Green tones, rounded outline cards, and standard clean architecture principles.
- **Tech Stack:** Kotlin, Jetpack Compose, SQLite Room Database, manual dependency injection via custom containers, JSON backup/restore.
- **Highlights:** Dynamic search dock, staggered grid reflow layout, rich markdown-helper formatting editor toolbar, and SAF (Storage Access Framework) backups.
