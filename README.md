### 🍳 Featured Project: [Freezer-to-Feast](https://github.com/anuraj07/FreezerToFeast)

> **A premium, serverless Android application designed with a "Quiet Luxury" aesthetic.** It helps users track kitchen parameters, customize dietary choices, log daily meals, and leverage client-side Generative AI (`gemini-2.5-flash`) via the Firebase Vertex AI SDK to suggest tailored recipes based on available ingredients.
